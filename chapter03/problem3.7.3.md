# Problem 3.7.3

(a) If $\mathbf{A}$ contains a zero row or zero column, then
$rank\left(\mathbf{A}\right) < n$.

(b) The reduced form of $\mathbf{A}$ will have a zero row, and so
$rank\left(\mathbf{A}\right) < n$.

(c) The reduced form of $\mathbf{A}$ will have a zero column, and so
$rank\left(\mathbf{A}\right) < n$.
